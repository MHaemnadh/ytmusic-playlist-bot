# 🎵 YouTube Playlist Automation (Telugu Songs)

This project automatically:
- Creates a YouTube playlist
- Searches songs
- Adds them automatically
- Avoids duplicates
- Resumes progress daily

## 🚀 Features
- OAuth login (secure)
- No duplicate songs
- Resume support
- Quota-safe (80 songs/day)

## ⚙️ Setup

1. Install dependencies:
pip install -r requirements.txt

2. Add your `client_secret.json`

3. Run:
python youtube_playlist.py

## 📁 Files
- songs.txt → list of songs
- done.txt → completed songs
- playlist_id.txt → playlist reference
